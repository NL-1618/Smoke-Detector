ผลงานชิ้นนี้จัดทำโดย นางสาวนิษฐเนตร์ พ่วงมีชัย
แรงบันดาลใจของผลงานชิ้นนี้มาจาก การสูบพอด บุหรี่ในโรงเรียนซึ่งเป็นเรื่องที่ผิด จึงได้เกิดแนวคิด Smoke Detector จากนั้นจึงได้ศึกษาหาข้อมูลโค้ดและอุปกรณ์ที่ใช้ จากhttps://shorturl.at/jCGPv
และได้นำมาทดลอง แต่ก็ยังไม่ประสบความสำเร็จ ลองผิดลองถูก หาวิธีแก้ไขปัญหา แก้ไขโค้ดอยู่หลายครั้ง เช่น เชื่อมเซนเซอร์กับWifiไม่ได้ หาพอร์ทไม่เจอ จนออกมาเป็นชิ้นงานนี้
แต่ ณ ปัจจุบันการตั้งค่าเกณฑ์ (threshold) ของเซนเซอร์ควัน ยังต้องมีการพัฒนาเพื่อหาค่าที่แม่นยำและเสถียรสำหรับใช้งานจริง
